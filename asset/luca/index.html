Luca & Friends Move Learning

A Learning and AR game for kids
Apple App Store Google Play

About

Luca & Friends Move Learning is a educational game that provide a AR enviroment for kids to learn. It make use of AI computer vision to track the movement of the user and combine with the AR environment to provide a fun and interactive experience.

Computer Vision

One of the biggest challenges of this project is to make the computer vision work. To do so we use a Google powered SDK Media-pipe to capture user movement. In partictular, we use Pose model to capture some key landmarks of the user, and combine with the gameplay to simulate the AR enviroment.

Firebase

The app is built using Firebase as a backend. With Firebase, the data is stored in a cloud database. Users can create their account and login. Their account can also link to Google, Facebook or their email in order to login at a different device. Other user data such as playing progress and achievements are stored in the database. For parents to estimate their children's progress.
