Project

Speaking Dead

Speech Recognition mobile game

About

Speaking Dead is a speech recognition game that use voice as input to play. User who speaks a correct word will count for 1 score. It internally provides a speech recognition libaray so it can support the recognition feature without internet connection. click here to watch a video game play

How it built

The game was developed using Unity. Unity handles the game play such as animation, handling voice inputs, counting scores and language switching etc. The recognition part is powered by a compiled libaray that imports as a plug-in. As a result, voice inputs received from Unity could be passed to the library to retrive text outputs.

Recognition

Since the recognition is done by compiled library, there are APIs provided to call a desired function. It would require users to turn on microphone function when users first open the game. During the game, it keeps recording and sending voice data to the compiled library in order to get recognized text result.

Update libaray

Speaking dead will auto update the recognition library when needed. If there are network connection when the app opens, it will compare the checksum of latest library in remote server and current library in local. If there are any differences, it will download the verison from remote server.
